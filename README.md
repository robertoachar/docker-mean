> A playground for Docker with MEAN stack.

![Markdown Course](preview/docker-mean.gif)

# Development

* Cloning the repo

```bash
$ git clone https://github.com/robertoachar/docker-mean.git
```

* Installing dependencies

```bash
$ npm install
```

# Docker

* Build image

```bash
$ docker-compose build
```

* Start containers

```bash
$ docker-compose up
```

* Stop containers

```bash
$ docker-compose down
```
